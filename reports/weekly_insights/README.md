# Weekly Insights Data

This directory stores daily trend snapshots used to generate the Sunday weekly insights newsletter.

## Data Files

- `{YYYY-MM-DD}_{segment}.json` - Daily trend snapshots for each segment
- Files are auto-generated by the daily pipeline
- Files older than 10 days are auto-cleaned

## Usage

The daily newsletter pipeline saves trend data here, and the Sunday weekly insights workflow reads from these files to synthesize the week's insights.
